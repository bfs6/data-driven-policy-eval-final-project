2020 Census Redistricting Data (P.L. 94-171) County Shapefile for Missouri

##Redistricting Data Hub (RDH) Retrieval Date
02/25/2021

##Sources
Shapefiles for the 2020 Census Redistricting Data (P.L. 94-171) are available for download from https://www.census.gov/geographies/mapping-files/time-series/geo/tiger-line-file.html
The Missouri County file direct download URL is https://www2.census.gov/geo/tiger/TIGER2020PL/LAYER/COUNTY/2020/tl_2020_29_county20.zip

##Fields
    Field Name                                                                             Description
0    STATEFP20                                                             2020 Census state FIPS code
1   COUNTYFP20                                                            2020 Census county FIPS code
2   COUNTYNS20                                                            2020 Census county GNIS code
3      GEOID20  County identifier; a concatenation of 2020 Census state FIPS code and county FIPS code
4       NAME20                                                                 2020 Census county name
5   NAMELSAD20       2020 Census name and the translated legal/statistical area description for county
6       LSAD20                          2020 Census legal/statistical area description code for county
7    CLASSFP20                                                             2020 Census FIPS class code
8      MTFCC20                                                    MAF/TIGER feature class code (G4020)
9      CSAFP20                                              2020 Census combined statistical area code
10    CBSAFP20            2020 Census metropolitan statistical area/micropolitan statistical area code
11  METDIVFP20                                                  2020 Census metropolitan division code
12  FUNCSTAT20                                                           2020 Census functional status
13     ALAND20                                                                   2020 Census land area
14    AWATER20                                                                  2020 Census water area
15  INTPTLAT20                                              2020 Census latitude of the internal point
16  INTPTLON20                                             2020 Census longitude of the internal point

##Processing
The shapefile was retrieved from the download link (see sources above) and was renamed using RDH conventions with a python script.
